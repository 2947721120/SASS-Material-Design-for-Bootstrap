animation ❯ Smooth Scrolling

Smooth scrolling allows you to link to other content in the same page and create a smooth scroll effect.

We automatically offset the content depending on whether there is an App Bar and Tool Bar within it.

Smooth Scrolling uses the velocity.js package to create a high performance scroll effect.

You can add easing and duration using the data-attributes data-scroll-duration and data-scroll-easing on the anchor element. The duration is in milliseconds and the easing should be one of Velocity's easing functions (unless you have crested your own). The default duration is 1000 milliseconds and the default easing is easeInOutQuint

<ul class="list-inline">
	<li><a href="#one">Content Item 1</a></li>
	<li><a href="#two" data-scroll-duration="2000">Content Item 2</a></li>
	<li><a href="#three" data-scroll-duration="3000" data-scroll-easing="spring">Content Item 3</a></li>
</ul>
