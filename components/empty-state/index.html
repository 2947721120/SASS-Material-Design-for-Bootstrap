components ❯ Empty State

An empty state is presented to the user when there is no content available

Content inside the empty state is automatically centered for you.

Basic

<div class="empty-state">
	<i class="icon icon-xxl material-icons">done_all</i>
	<h3>You don't have any content here</h3>
	<p>Why not add a <a class="btn btn-success">call to action</a> so that you can change that?</p>
</div>
