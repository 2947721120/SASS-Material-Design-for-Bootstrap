Material Design Form

To add floating labels to a text input or textarea, you need to do two things:

  1. Don't include the standard label
  2. Add a material class to the form-group

So this:

<div class="form-group">
	<label>Email address</label>
	<input type="text" class="form-control" placeholder="Your Name">
</div>

Becomes this:

<div class="form-group material">
	<input type="text" class="form-control" placeholder="Your Name">
</div>
