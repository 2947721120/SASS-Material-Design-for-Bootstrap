When writing colours in SASS, there is a utility function to get the colour value. 
element {
	background-color: get-color(blue, 600);
	color: get-color(pink, 50);
}

Background Colours

Add the following classes to any html element to apply a background colour. The color for each one is also set to black or white.

Black .bg-black White .bg-white
