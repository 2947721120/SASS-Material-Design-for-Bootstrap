We use the ramjet library to add authentic motion to the page.

You need two elements to make a motion work - the trigger and target.

The trigger should have a .morph-trigger class and the target should have a .morph-target class.

<a id="a" class="btn btn-info morph-trigger" data-morph-target="b">Eat Me</a>

<div id="b" class="paper morph-target">

	<a class="close morph-dismiss"><i class="icon material-icons">close</i></a>

	<h4>Hey there</h4>

</div>
Eat Me
close

Hey there

By adding data-morph-hide="true" when we transition we will remove the trigger when the target is displayed.

<a id="a" class="btn btn-info morph-trigger bg-red bg-50" data-morph-hide="true" data-morph-target="b">Drink Me</a>

<div id="b" class="paper morph-target bg-purple bg-300">

	<a class="close morph-dismiss"><i class="icon material-icons">close</i></a>

	<h4>Refreshing</h4>

</div>
Drink Me
close

Refreshing

To use one element as a trigger, but morph from another element to the target use the data-morph-trigger-from="" attribute and add the element's id.

<div id="e" class="bg-blue bg-50">
	<p>Sleep</p>
	<p><a class="btn btn-primary morph-trigger" data-morph-hide="true" data-morph-trigger-from="e" data-morph-target="f">Now</a></p>
</div>

	<a class="close morph-dismiss">
		<i class="icon material-icons">close</i>
	</a>

	<h4>zzzZZZzZZZZZZZzzz</h4>

</div>
