Controls

Switch

Add a .switch class to any checkbox element to turn it into a switch.

To customise the colour of the switch, use the primary colour name. We do not offer hues for colouring as the background "rail" is based off the primary.

<div class="switch">
	<input type="checkbox" name="switch1Value" id="switch1" checked>
</div>

<div class="switch switch-pink">
	<input type="checkbox" name="switch2Value" id="switch2" checked>
</div>

Please note, we initialise the switch control via javascript, so it is important you do not add a <label> to your wrapper!
