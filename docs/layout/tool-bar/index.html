--- layout: default title: Tool Bar ---

Tool Bar's allow you to add extra levels of controls to your page. Commonly they are used above the App Bar but can be used anywhere on a page.

To add a tool bar to above your app bar, you need to add a .has-tool-bar class to your containing app bar:

<nav class="app-bar has-tool-bar ...

Then add the tool bar inside the container at the top.

<nav class="app-bar has-tool-bar ...
	<div class="tool-bar bg-blue-grey bg-50">
		...
	</div>

Tool Bar's are attached to other elements, such as Paper.
