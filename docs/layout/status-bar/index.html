The Status Bar is a very compact area (25px high) which is fixed to the top of the page so you should use it with discretion. Whilst an operating system will usually provide it's own status bar, you may want to add one to your page.

The status bar, by default, is fixed to the top of the page and has an opacity of 0.8.

When displaying a status bar, make sure you add .has-status-bar class to the <body> element. This adds the correct padding to other elements on the page.

<body class="has-status-bar">

	// body content

</body>
