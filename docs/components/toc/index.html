A Table of Contents (ToC) allows you to display a list of links to anchor points within the page, similar to the one on the Material Design site.

You can use the primary colour name to style your ToC, such as <div class="toc toc-cyan">

To add a header, user an <h3> tag and to add content to the ToC, use a vanilla <ul>.

<div class="toc toc-cyan">
	<h3>Example Header</h3>
	<ul>
		<li><a href="#one">Content Item 1</a></li>
		<li><a href="#two">Content Item 2</a></li>
		<li><a href="#three" data-scroll-duration="3000" data-scroll-easing="spring">Content Item 3</a></li>
	</ul>
</div>
