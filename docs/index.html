--- layout: default title: Material Design for Bootstrap ---

Material Design is a set of design principles that Google developed in 2014.

Bootstrap is one of the most popular CSS / JS Frameworks on the web.

Material Design for Bootstrap joins the two.

We use the power of Bootstrap and adapt it to Material Design principles.

Furthermore, Material Design introduced some new UI elements which this library brings to Boostrap.

Some things are the same, some are better and some are new.

Quick Start

Including this library in your project is as simple as replacing the original Bootstrap files with these ones:

<link href="https://cdn.rawgit.com/band-x-media/SASS-Material-Design-for-Bootstrap/master/dist/material-bootstrap.min.css" rel="stylesheet">
<script type="text/javascript" src="https://cdn.rawgit.com/band-x-media/SASS-Material-Design-for-Bootstrap/master/dist/material-bootstrap.min.js"></script>
