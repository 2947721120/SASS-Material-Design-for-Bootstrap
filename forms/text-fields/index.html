Floating Labels

To add floating labels to a text input or textarea, you need to do two things:

  1. Don't include the standard label
  2. Add a material class to the form-group

So this:

<div class="form-group">
	<label>Email address</label>
	<input type="text" class="form-control" placeholder="Your Name">
</div>

Becomes this:

<div class="form-group material">
	<input type="text" class="form-control" placeholder="Your Name">
</div>

Full Width Inputs

Full-width text fields are useful for more in-depth tasks.

<div class="form-group material full-width">
	<input type="text" class="form-control" placeholder="Subject">
</div>

Multi Line Text Field

Use a textarea as you normally would, but add an autogrow class and add the row attribute to the element.

<div class="form-group material">
	<textarea rows="1" placeholder="Write a Message" class="form-control autosize"></textarea>
</div>

Adding Icons

To add icons, you need to:

  1. Add a .with-icon class to the containing .form-group
  2. Add an icon above the input.

If you want to add the icon to the right of the input field, add an icon-right class

<div class="form-group material with-icon">
	<i class="icon material-icons">person</i>
	<input type="text" class="form-control" placeholder="What's your name">
</div>
<div class="form-group material with-icon">
	<i class="icon material-icons icon-right">info</i>
	<input type="number" class="form-control" placeholder="Choose A Number">
</div>

Character Limit

If you want to restrict the number of characters an input form can accept, you should:

  1. Add a character-count to the input / textarea field.
  2. Add Parameters using the data attribute:
    • data-cc-limit: the total number of characters allowed
    • data-cc-low: the number of characters left at which point we warn the user
<div class="form-group material">
	<input type="email" class="form-control character-count" data-cc-limit="10" data-cc-low="2" placeholder="What's your email">
</div>
<div class="form-group material">
	<textarea class="form-control character-count" placeholder="Write a Message" data-cc-limit="100" data-cc-low="10"></textarea>
</div>
